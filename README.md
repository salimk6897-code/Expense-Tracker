# Pocket Mate 💰
### The Ultimate Privacy-First Splitwise Alternative

Pocket Mate is a high-performance Android application designed for seamless personal expense tracking and group debt management. Built with **Jetpack Compose** and **Room Database**, it follows modern Android development best practices.

## 🚀 Key Features
- **Elite Split Engine**: Automatically calculates the minimum number of transactions to settle group debts.
- **Privacy-First**: All data is stored locally on your device using an encrypted Room Database. No cloud required.
- **Universal Currency Support**: Full support for INR (₹), USD ($), EUR (€), and GBP (£) across all screens.
- **Smart Dashboard**: A premium, eye-catching UI with a real-time spending summary.
- **Transparency**: Detailed breakdown for every expense showing exactly who paid and who owes what.
- **Safe Mode**: Deletion warnings and confirmation dialogs to prevent data loss.

## 🛠 Tech Stack
- **Language**: Kotlin
- **UI Framework**: Jetpack Compose (Material 3)
- **Database**: Room Persistence Library
- **Architecture**: MVVM (Model-View-ViewModel)
- **Dependency Management**: Gradle Version Catalog (libs.versions.toml)
